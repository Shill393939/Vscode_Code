# 环形链表

## 155

```cpp
    /**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    bool hasCycle(ListNode *head) {
            ListNode *fast = head;//这里要注意啊 初始化的时候是要给个头节点的 就因为这里没记住，看了好几遍
            ListNode *slow = head;
            while(fast!=NULL&&fast->next!=NULL)
            {
                slow = slow->next;
                fast = fast->next->next;

                if(fast==slow)
                {
                    return true;
                }
            }
            return false;
    }
};
```

## 原理就是 一个快一个慢 如果是环的话，一定会相遇
